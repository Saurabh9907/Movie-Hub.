# 🎬 Movie-Hub - Movie Search Web Application

Movie-Hub is a responsive movie search web application built using HTML, CSS, and JavaScript. It integrates the OMDb API to provide real-time movie search results and detailed information about movies.

## 🚀 Features

- Search movies by title
- Real-time search suggestions
- View movie poster
- Display release year
- View movie rating
- Show genre
- Display writer and cast
- Read movie plot
- View language and awards
- Responsive user interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OMDb REST API
- Fetch API
